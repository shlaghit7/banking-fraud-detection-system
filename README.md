## 🚀 Overview

This project simulates a real-world banking system designed to track transactions and detect fraudulent activities using SQL, Power BI, and Machine Learning.

---

## 🎯 Objective

To build an end-to-end data analytics solution that identifies suspicious transactions and provides actionable insights for fraud prevention.

---

## 🧰 Tech Stack

* SQL (MySQL/PostgreSQL)
* Power BI
* Python (Scikit-learn, Pandas)

---

## 📊 Features

* Normalized relational database (Customers, Accounts, Transactions)
* Rule-based fraud detection using SQL
* Machine Learning anomaly detection (Isolation Forest)
* Interactive Power BI dashboard (Deloitte-style UI)
* 1000+ simulated transaction dataset

---

## 🗂️ Database Schema

* Customers
* Accounts
* Transactions
* Branches
* Fraud Alerts

---

## 🚨 Fraud Detection Logic

* High-value transactions
* Rapid transaction frequency
* Location anomalies
* ML-based anomaly detection
